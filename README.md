# Zahra Dhaine — CV Website

This is a lightweight, modern CV website (HTML/CSS/JS) ready for **GitHub Pages**.

## 1) Run locally
Just open `index.html` in your browser.

If you want a local server (recommended for best behavior):
- VS Code → install **Live Server** extension → right click `index.html` → **Open with Live Server**

## 2) Publish on GitHub Pages (quick steps)
1. Create a new GitHub repository (example: `zahra-cv`)
2. Upload all files from this folder to the repo root
3. In GitHub: **Settings → Pages**
4. Source: `Deploy from a branch`
5. Branch: `main` and Folder: `/ (root)`
6. Save — GitHub will give you a URL like:
   `https://USERNAME.github.io/REPO/`

## 3) Generate QR code
- Open the website
- In the **QR Code** section, paste the GitHub Pages URL
- Click **Generate**
- Click **Download QR**

## Edit content
Open `index.html` and update any text, skills, or experience.
